Rules for writing functions

  • Your function name must be unique
  • The function must accept a single string as an input.
  • The value returned by the function must be a string.
  • Do not enter comments
  • Do not make functions supporting your function, outside the function.
  • Use only single quotes for addressing strings or string like constructs.

Suggestions

  • Use the same notation as the inbuilt functions
  • Use compress/prettify while storing/modifying functions accordingly. This improves the loading time and the space consumed

Example:

    var encodeURIFn = function(uri){
      return encodeURIComponent(uri);
    };
    var helloFn = function(uri){
      return ('hello '+uri);
    };
